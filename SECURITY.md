# Security Policy

## Supported Content

This repository is actively maintained, but payloads are provided for educational and authorized testing purposes only.

<br>

## ⚠️ Notice

Some payloads may:
- Modify system behavior
- Access network resources
- Use webhooks or external services

<br>

##  🚨 Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Instead, report it privately to me.

I will review and address security concerns as quickly as possible.
