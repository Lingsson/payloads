# Code of Conduct

## Purpose

This project is focused on educational and authorized security testing tools, including payloads and automation scripts.


---

## Expected Behavior

All contributors are expected to:

- Be respectful and professional
- Follow the contribution guidelines
- Submit only authorized and clearly documented payloads
- Avoid spam or low-quality submissions
- Respect the purpose of this project

---

## Unacceptable Behavior

The following is not allowed:

- Harassment, insults, or abusive behavior
- Spam or intentionally misleading contributions
- Attempting to disrupt systems or services without permission

---

## Enforcement

Maintainers may:
- Remove or reject contributions
- Close issues or pull requests
- Block users who violate these rules
